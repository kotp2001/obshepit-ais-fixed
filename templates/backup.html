Как работает резервное копирование
Создание копии

Нажмите «Создать резервную копию» → сразу скачайте файл на компьютер. Копия содержит все данные: заказы, меню, пользователей, отчёты. Файлы на сервере удаляются при перезапуске!

Восстановление (сайт работает)

Нажмите «Восстановить» рядом с нужной копией. Все текущие данные будут заменены данными из копии. Сделайте новую копию перед восстановлением.

Восстановление через DBeaver (если сайт недоступен)

Если сайт не открывается — восстановите данные напрямую в базу данных:

  1. Откройте DBeaver → подключитесь к obshepit_db_final
  2. SQL Editor → вставьте содержимое скачанного .sql файла → нажмите F5
  3. После выполнения перезапустите сервис на Render.com → Manual Deploy
